Gmail

Open this page in Chrome → select the entire dark signature card → copy → go to Gmail Settings → General → Signatures → paste.

Outlook (Windows)

File → Options → Mail → Signatures → New → click into the editor → Insert → Object → HTML. Alternatively, open this page in a browser, select the signature, copy, and paste into the Outlook signature editor directly.

Outlook (Mac) / Apple Mail

Open this page in Safari → select the signature card → copy → paste into Mail's signature editor (Mail → Preferences → Signatures).
